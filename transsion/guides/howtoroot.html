How to Root your device!

Root Method: Magisk

Why would i want Root?

Root is basically obtaining Super User Permission to do pretty much anything at all in your Android device. From system-level debloating to major UI modifications and so on. Rooting has perks, lots of them, but it also has it's downsides such as voiding your warranty, loss of safetynet and playintegrity and blocked access to banking applications. These problems do have workarounds but they don't work all the time on all the devices.

Where do i start?

YOU WILL NEED A COMPUTER!

YOUR BOOTLOADER MUST BE UNOCKED!

  • First off, download the latest stable magisk from the Repository
  • Now you will need the boot.img of your device which can be by downloading your device firmware.
  • Put the boot.img file into your device and launch the Magisk application.
  • Tap the "Install" button and select file for patching
  • Search for your file and tap it.
  • Once the patching process is complete, copy the file to your Computer along with the vbmeta.img located also in your firmware. This will be used to disable AVB (Android Verified Boot).
  • On your computer, put both image files in a folder and open your terminal in that folder.
  • Connect your device and reboot to bootloader
  • You must run the AVB disabling command before and after the boot image flashing command to avoid bootlooping
  • run fastboot --disable-verity --disable-verification flash vbmeta vbmeta.img to disable AVB
  • Now run fastboot flash boot (patchedboot).img
  • Run the AVB command again and then, fastboot reboot

With all that done, you should be booted with Magisk properly installed.
