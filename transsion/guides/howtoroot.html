How to Root your device!

Root Method: Magisk

Why would i want Root?

Root is basically obtaining Super User Permission to do pretty much anything at all in your Android device. From system-level debloating to major UI modifications and so on. Rooting has perks, lots of them, but it also has it's downsides such as voiding your warranty, loss of safetynet and playintegrity and blocked access to banking applications. These problems do have workarounds but they don't work all the time on all the devices.

Where do i start?

YOU WILL NEED A COMPUTER!

YOUR BOOTLOADER MUST BE UNOCKED!

DO NOT SKIP STEP 2 AND 3!

  1. First off, download the latest stable magisk from the Repository
  2. Now you will need the boot.img of your device which can be by downloading your device firmware.
  3. Also in your firmware is the vbmeta.img file which is REQUIRED to DISABLE VERITY CHECKS/BOOT VERIFICATION!
  4. Put the boot.img file into your device and launch the Magisk application.
  5. Tap the "Install" button and select file for patching
  6. Search for your file and tap it.
  7. Once the patching process is complete, copy the file to your Computer along with the vbmeta.img located also in your firmware. This will be used to disable AVB (Android Verified Boot).
  8. On your computer, put both image files in a folder and open your terminal in that folder.
  9. Connect your device and reboot to bootloader
  10. You must run the AVB disabling command before and after the boot image flashing command to avoid bootlooping
  11. run fastboot --disable-verity --disable-verification flash vbmeta vbmeta.img to disable AVB
  12. Now run fastboot flash boot (patchedboot).img
  13. Run the AVB command again and then, fastboot reboot

With all that done, you should be booted with Magisk properly installed.

Issues and their Solutions with Magisk (FAQs)

  • I updated Magisk and now zygisk doesn't work = Uninstall/Restore previous "zygisk compatible" version of magisk app and repatch stock boot image and repeat the flashing process
  • Help!, i flashed a newly patched image but my device doesn't boot = ALWAYS run the disable verity command before and after flashing a patched boot image for safety measures!
